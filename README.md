# QR Code Generator — Veolia

## Hebergement gratuit sur Render.com

1. Cree un compte sur https://render.com
2. Clique "New Web Service"
3. Connecte ton GitHub et uploade ces fichiers
4. Render detecte automatiquement le requirements.txt
5. Start command : `python app.py`
6. Partage le lien avec tes collegues

## Fichiers
- app.py          : serveur Flask (backend)
- index.html      : interface web (frontend)
- requirements.txt : dependances Python
